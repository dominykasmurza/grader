# Portfolio description

**Grader — Python/OpenCV, 2026**

Developed a batch optical mark recognition and grading system for true/false examination sheets. Implemented ArUco-based perspective correction, fallback alignment with incomplete marker detection, ambiguity handling, configurable nonlinear scoring, annotated quality-control output, and detailed CSV reporting.
